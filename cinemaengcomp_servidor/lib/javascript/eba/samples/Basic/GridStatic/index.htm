EBA: Grid Control V2.7 Static Grid Demo

 This is an example of a static grid, or a grid without a remote datasource. In this case, the data is contained in XML embedded directly in the HTML.

In this instance, the resulting XML could be deposited into a hidden form field and posted with a regular form, but saving and loading has not been enabled here.

This demo could be used as a grid template, as each input type is demonstrated here. Notice the use of comboboxes, listboxes, checkboxes, textboxes, image cells, number fields, date boxes, and hyperlink fields.

Press the INSERT key to create rows.
Press the DELETE key to delete rows.
