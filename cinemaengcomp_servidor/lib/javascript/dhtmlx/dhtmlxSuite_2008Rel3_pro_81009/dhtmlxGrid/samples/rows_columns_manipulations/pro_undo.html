Sample: dhtmlxGrid Undo/Redo dhtmlxGrid main page

Just edit anything , and you can use UNDO | REDO buttons to restore previous state.

 

 
    <script>
 
        mygrid = new dhtmlXGridObject('gridbox');
        ....
        mygrid.enableUndoRedo();
        ...
        mygrid.doUndo()
        mygrid.doRedo()
</script>
