Sample: dhtmlxGrid Event Handler for onRowDblClicked dhtmlxGrid main page

You can set handler for onRowDblClicked event


 
<div id="gridbox" width="100%" height="250px" style="background-color:white;overflow:hidden"></div>
<script>
 
    function doOnRowDblClicked(rowId){
        return confirm("Row with ID "+ rowId + " was double clicked. Do you want to proceed?")
    }
    mygrid = new dhtmlXGridObject('gridbox');
    ...
    mygrid.attachEvent("onRowDblClicked",doOnRowDblClicked);
    ...
    mygrid.init();
    mygrid.loadXML("grid.xml");
</script>
