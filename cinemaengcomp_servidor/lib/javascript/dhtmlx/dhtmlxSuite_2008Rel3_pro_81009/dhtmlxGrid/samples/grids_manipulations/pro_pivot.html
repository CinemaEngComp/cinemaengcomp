Sample: dhtmlxGrid Pivot tables dhtmlxGrid main page

Pivot table allows to quickly create crosstabs.
You can set it instead of the grid or separately and then choose fields and operation. 

 
    mygrid.makePivot();  //create pivot
    mygrid.makePivot('gridboxP');  //create pivot in external container
    mygrid.hidePivot();  //hide pivot
    mygrid.makePivot(,{column_list:[null,'Title','Author','Money']}); //pivot with limited set    
    //pivot where action (1st in list) by value (4th column) is calculated between second and third columns 
    mygrid.makePivot('gridboxP',{x:1,y:2,value:3,action:0});
