Sample: dhtmlxGrid Extended drag-n-drop dhtmlxGrid main page

Try to move rows from one grid to another and you wiil see special signs with the cursor:

  • allowed/denied drag-and-drop
  • how many rows (top nodes for treegrid) you are dragging

    • Extended drag-n-drop can be set by including dhtmlx_extdrag.js
    Allow mercy drag-and-drop Allow mercy drag-and-drop

       mygrid = new dhtmlXGridObject('gridbox');
    ...
        mygrid.enableMercyDrag(true);
