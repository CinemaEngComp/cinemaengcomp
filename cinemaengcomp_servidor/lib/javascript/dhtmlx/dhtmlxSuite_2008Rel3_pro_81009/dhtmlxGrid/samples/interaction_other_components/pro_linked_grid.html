Sample: dhtmlxGrid Grid linked to grid dhtmlxGrid main page

You can use one grid as editor for another grid. Just try to edit column with authors.

 
 
<div id="gridbox" width="100%" height="250px" style="background-color:white;overflow:hidden"></div>
<script>
 
    subgrid = new dhtmlXGridObject('gridbox_sub');
    ...
    subgrid.loadXML("authors.xml",function(){
        mygrid = new dhtmlXGridObject('gridbox');
        ...
        mygrid.setSubGrid(subgrid,2,0);  //link subgrid to second column (2) and use value of column (0) as label
        mygrid.init();
        mygrid.loadXML("grid.xml");
    });
</script>
