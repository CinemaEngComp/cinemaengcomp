Sample: dhtmlxGrid Grid linked to tree dhtmlxGrid main page

Tree can be used as editor for the grid. Try to edit column with authors.

 
 
<div id="gridbox" width="100%" height="250px" style="background-color:white;"></div>
<div id="treeboxbox_tree" style="width:300px;height:200px"></div>
<script>
 
    tree=new dhtmlXTreeObject("treeboxbox_tree","100%","100%",0);
    ...
    tree.loadXML("tree.xml",function(){
        mygrid = new dhtmlXGridObject('gridbox');
        ...
        mygrid.setColTypes("dyn,ed,stree,price,ch,co,ra,ro");
        mygrid.setSubTree(tree,2,0);
        mygrid.init();
        ...
    });
</script>
