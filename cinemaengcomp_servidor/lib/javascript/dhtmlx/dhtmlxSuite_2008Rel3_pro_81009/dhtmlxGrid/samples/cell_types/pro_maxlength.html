Sample: dhtmlxGrid Using Event Handlers to limit cell editor max length dhtmlxGrid main page

You can set limitation for the entered text length when cell is edited.
Try to edit some cell and print more then ten symbols.

Column with index 1 (second column in grid) has limitation set on value length with the help of onEditCell event handler. In current sample maximum length of the velaue is 20 symbols. 


 
This code limits max length of second column (column with index 1) cell editor value to 20 symbols:
 
mygrid.attachEvent("onEditCell",function(stage,rowId,cInd){
    if(cInd==1 && stage==1){
        mygrid.editor.obj.onkeypress = function(e){
            if(this.value.length>=20){
                return false;
            }
        }
    }
    return true;
})
