Sample: dhtmlxGrid Date/Decimal format dhtmlxGrid main page

You can set Date and Decimal number formats using setDateFormat and setNumberFormat methods

 
<div id="gridbox" width="100%" height="250px" style="background-color:white;overflow:hidden"></div>
<script>
 
    mygrid = new dhtmlXGridObject('gridbox');
    mygrid.setColTypes("edncl,ednro,calendar");
    ...
    mygrid.setNumberFormat("%script%0.00",1);
    mygrid.setNumberFormat("0,000",0);
    mygrid.setDateFormat("m-d-y"); 
    mygrid.init();
    mygrid.loadXML("grid_format.xml");
</script>
