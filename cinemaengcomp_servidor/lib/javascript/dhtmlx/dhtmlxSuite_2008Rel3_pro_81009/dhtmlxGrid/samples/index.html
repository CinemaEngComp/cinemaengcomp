dhtmlxGrid 2.0. Samples

Featured Samples:

  • Save Data dynamicaly
    • Online sample demonstrating grid connection to server side datasource. You can add/delete/update records - all changes will be saved to database (your changes are available for you within php session time frame)
  • Drag and Drop between Grids
    • Try dhtmlxGrid drag-n-drop capabilities with this sample.
  • Attaching Event Handlers
    • Possibility to control events on different stages of control lifecycle is a basis for successful programming. CUrrent sample demonstrates basic abilities. For complete list of possible events aply to API documentation
  • Grouping rows by column
    • Grouppng is new functioality for dhtmlxGrid v.1.5. Try it and you'll see how flexible it is and how useful it can be for users of your application.
  • Grid with subgrid
    • Starting from v.1.5 grid can contain independent subgrids in it. This possibility can be used to include nested data with entirely new structure into grid.
  • Subrow excell
    • Subrows is another new functionality for dhtmlxGrid which is available since v.1.5. Can be used for showing some additional row related block of information.
  • Grid in split mode
    • Table views splitted in scrollable parts are useful for displaying data where some part should stay visible while the other part is scrolled.
  • Complex content in header
    • dhtmlxGrid can have more than one row in header and these rows can be used to place content which is more complex than just a column lable.
  • dhtmlxGrid & dhtmlxTree interaction
    • dhtmlx components can interact with each other. This can be quite useful for modern web applications. Drag records from tree and drop it into grid and vice versa.
  • 50,000 records in grid
    • This sample demonstrates how fast dhtmlxGrid works with big number of records. 50,000 records is not the limit but enough to size this up. Load, filter and sort 50,000 records with this sample.
  • 50,000 records in grid with paging
    • Try ne paging with non-linear buffering (since v.1.6) with 50,000 records and assess performance and new visual appearence of paging.
  • Move column
    • Not only rows can be moved in dhtmlxGrid. Columns also. Try changing columns position with this sample.
  • Math calculations
    • There is a support for different mathemetical operations in dhtmlxGrid. You can specify formulas with script or in XML.
