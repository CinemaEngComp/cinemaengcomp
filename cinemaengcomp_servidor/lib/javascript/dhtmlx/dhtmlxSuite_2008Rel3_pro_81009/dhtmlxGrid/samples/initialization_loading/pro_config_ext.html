Sample: dhtmlxGrid Run commands from XML dhtmlxGrid main page

Run methods before grid intialization

  • Example: Load grid with xp skin set from XML by calling setSkin method before grid initialization

    •  
<rows>
    <head>
        <beforeInit>
            <call command="setSkin"><param>xp</param></call>
        </beforeInit>
        ...
    </head>
    Same as with script: 
     
    mygrid = new dhtmlXGridObject('gridbox2');
    ...
    mygrid.setSkin("xp");
    mygrid.init();

    Run methods after grid intialization

  • Example: Load grid and set 2nd column hidden by calling setColumnHidden method from xml after grid initialization

    •  
<rows>
    <head>
        <afterInit>
            <call command="setColumnHidden"><param>1</param><param>true</param></call>
        </afterInit>
        ...
    </head>
    Same as with script: 
     
    mygrid.setColumnHidden(1,true);
