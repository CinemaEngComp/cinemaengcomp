Sample: dhtmlxGrid Setting tab order dhtmlxGrid main page

Grid supports setting specified tab order fir each column.
Just set method "setTabOrder" after grid initialization and pass list of tab orders into it

 
 
    mygrid = new dhtmlXGridObject('gridbox');
    ...
    mygrid.setColTypes("dyn,ed,ed,price,ch,co,ra,ro");
    mygrid.init();
    
    mygrid.setTabOrder("1,3,5,3,4,6,7,8")
    ...
