dhtmlxTreeGrid 2.0. Samples

Featured Samples:

  • Drag and Drop between Grids
    • Rows can be dropped from TreeGrid to Grid and vica versa. Try this out with this sample.
  • Math calculations
    • If you need some mathemeical calculations this sample will make you see that dhtmlxTreeGrid provide such capability.
  • Aggregation in TreeGrid
    • Use autocalculated values to do usual calculations in treegrid footer
  • Search and Filtering in TreeGrid
    • Try to find necessary data in treegrid using built-in filters and searc capabilities
  • Dynamic loading (using PHP)
    • Sample demonstrats dynamical loading of data into dtmlxTreeGrid. This technic helps increasing overal performance for nested structures.
  • Paging in dhtmlxTreeGrid
    • Since v.1.6 dhtmlxTreeGrid supports paginal output. This is common way for solving performance problems with big datasets. But try Smart Rendering also.
  • Smart Parsing
    • Next to dhtmlxTree, dhtmxTreeGrid supports Smart XML Parsing for showing big structures. This technic increases overal performance for big trees without the necessity to split data into levels on server side. Just load data like you did before, dhtmlxTreeGrid will do everything else.
  • Smart Rendering in dhtmlxTreeGrid
    • Smart rendering is the technology which allows dhtmlxGrid users work with really huge datasets. Since v.1.6 it is available for tree structures also.
  • TreeGrid lines
    • By default dhtmlxTreeGrid had no visual connectors for nodes. Starting from v.1.5 you can choose either to turn connection lines for nodes on or off.
  • TreeGrid serialization
    • You can export dhtmlxTreeGrid content into XML string any time calling just one method.
  • TreeGrid Split Mode
    • Split dhtmlxTreeGrid content into two parts to have the ability to scroll them independently.
