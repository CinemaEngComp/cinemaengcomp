dhtmlxLayout 2.0. Samples

Featured Samples:

  • Attach dhtmlxTree to Layout
    • Since dhtmlx 2.0 you can attach any dhtmlx component to dhtmlxLayout panel with just single script command. See how easely dhtmlxTree can be attached in this sample. You also can see samples with Accordion, Grid, TreeGrid, Tabbar, Toolbar, Menu, Folders.
  • Attach Object to Layout
    • This sample demonstrates the ability to attach HTML objects to dhtmlxLayout panel. Just use any object from page and single javascript command. You also can see other samples demonstrating attaching dhtmlx components or external pages via URLs.
  • Expanding/Collapsing Cells
    • Expanding/collapsing panels is one of the main visual functionalities of dhtmlxLayout. Try it in action with this sample.
  • Layout Views
    • You can see all available predefined layout configurations in this sample. dhtmlLayout allows nested layouts, thus using combinations of different layouts you can get nearly any structure of interface.
  • Fullscreen Init
    • There are four ways you can initialize dhtmlxLayout object. This sample demonstartes full screen mode of initialization. You can also find samples of HTML object based, dhtmlxWindows based and dhtmlxLayout based.
  • Skinning
    • dhtmlxLayout is delivered with skins which can be inheritted by any dhtmlx 2.0 component attached to dhtmlxLayout. See available skins in this sample.
