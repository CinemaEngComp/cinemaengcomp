Sample: dhtmlxTree Loading from JSArray dhtmlxTree main page

There is a possibility to load data from JS Array

 reload from JS array(textarea)
load from external JS array data file 
 
    <script>
 
        var tree = new dhtmlXTreeObject('treeboxbox_tree', '100%', '100%', 0); 
        tree.setImagePath("../../codebase/imgs/csh_bluebooks/");
        tree.enableDragAndDrop(true);
        tree.loadJSArray([   [1,0,"1111"],
                               [2,0,"2222"],
                              [3,0,"3333"],
                             [4,2,"child"]    ])
        ....
        tree.loadJSArrayFile("data.js");
</script>
    
    Format of JS array
        [ID,parentID,text]
    order or rows in JS array doesn't matter
