Sample: dhtmlxTree Auto search dhtmlxTree main page

Just select any item in tree and start typing, tree item with name which you typed will be selected. To restart search use backspace button


 
            tree=new dhtmlXTreeObject("treeboxbox_tree","100%","100%",0);
            tree.setImagePath("../imgs/");
            tree.enableSmartXMLParsing(true);
            tree.enableDragAndDrop(true);
            //enable navigation
            tree.enableKeyboardNavigation(true);
            //enable auto search
            tree.enableKeySearch(true);
            tree.loadXML("tree.xml");
Select any item in tree and use next keys
  • Up arrow - select upper item
  • Down arrow - select lower item
  • Right arrow - open item
  • Left arrow - close item
  • Enter - call item's action
