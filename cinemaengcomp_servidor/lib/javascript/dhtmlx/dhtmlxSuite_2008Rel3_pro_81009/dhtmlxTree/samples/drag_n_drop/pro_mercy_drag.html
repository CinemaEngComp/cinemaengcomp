Sample: dhtmlxTree Copy with Drag-n-Drop dhtmlxTree main page

Mercy drag-and-drop allows you to copy nodes from one tree to another. In current sample it is enabled for the left tree.

 

 
<div id="treeboxbox_tree" style="width:200;height:200"></div>
<script>
 
        tree=new dhtmlXTreeObject("treeboxbox_tree","100%","100%",0);
        tree.setImagePath("../imgs/");
        tree.enableMercyDrag(true);     //enable mercy drag
        tree.loadXML("tree.xml")
</script>
