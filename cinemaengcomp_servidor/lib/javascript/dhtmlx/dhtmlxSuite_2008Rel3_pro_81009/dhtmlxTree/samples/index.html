dhtmlxTree 2.0. Samples

Featured Samples:

  • Text direction (LTR & RTL)
    • dhtmlxTree can work either with Left-to-Right text or with Right-to-Left text. Moreover, modes can be switched on the fly. This can be useful if you want to switch betwenn rtl and ltr languages without reloading complete page.
  • Pure text tree
    • Sample demonstrates capability of making tree as light as possible by turning off some or all images in tree.
  • Context menu based on item properties
    • dhtmlxMenu based context menu contains different items depending on properties of the tree node it was loaded for.
  • Updating server datasource
    • Sample demonstrates how to setup and use dhtmlxDataProcessor extension for dhtmlxTree for loading data from server and saving chnages back to server datasource.
  • Initialize from HTML
    • dhtmlxTree object can be initialized easily from html lists. No additional scripting needed.
  • Different loading modes
    • Starting from v.1.6 dhtmlxTree officially supports loading from JSON object. See it in action.
  • Smart XML parsing
    • Sample demonstrates how fast dhtmlxTree works with big trees when special technique of parsing data is used - Smart XML Parsing. No modifications for back end are necessary.
  • Smart Rendering in tree
    • Smart rendering is new feature for dhtmlxTree (available from v.1.6). It solves the problem with performance when tree structure contains levels with big number of nodes per level (500 and more).
  • Autoloading from XML
    • Autoloading (or dynamical loading) is another way for increasing performance for big trees. In its case data is loaded from server side on demand - when user expands level. Using this technique you can implement trees with really huge amount of data.
  • Editable items
    • Editable items is another popular dhtmlxTree feature. Sample demonstrates it in action.
  • Navigation and Edit with keyboard
    • dhtmlxTree supports standard keys for navigating data in tree and editing items. Also, you can set your own key maps.
  • Smart XML parsing and Sorting
    • Sample demonstrates the possibility to sort items in tree.
